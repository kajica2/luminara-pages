J = Eye (of Horus / Providence)
K = Crescent (Moon)
L = Sun (with 8 rays)
M = Arrow Up (Ascension)
N = Arrow Right (Direction)
O = Infinity (Lemniscate)
P = Spiral (Golden ratio)
Q = Chaos Star (8-pointed)
R = Merkaba (Star tetrahedron)
S = Vesica Piscis (sacred geometry)
T = Triskelion (Triple spiral)
U = Pyramid (with eye)
V = Chalice (Grail / Cup)
W = Sword (Blade / Will)
X = Key (Unlocking / Knowledge)
Y = Hourglass (Time / Saturn)
Z = Lightning (Power / Zeus)

NUMBERS → GEOMETRIC POLYGONS

0 = Decagon (10 sides)
1 = Dot (Point / Monad)
2 = Line (Dyad / Duality)

USAGE TIPS
