# LUMINARA Pages

Static, sovereign-futurism HTML pages for the LUMINARA universe. Deployed on Vercel.

## Structure

- `/` — landing index linking every page
- `/pages/` — the HTML pages (creator, index, light-spectrum, press, studio, vj, vysio, luminara-monolith, luminara-homepage-v1/v2)
- `/public/media/` — public assets (vysio demo webm)
- `/brand-kit/` — non-served: brand-kit.json, luminara-complete/ (fonts, svgs), luminara-video-pipeline.skill
